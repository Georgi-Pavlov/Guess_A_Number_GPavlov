# Guess_A_Number_GPavlov
This is an attempt for a simple console game "Guess a number" 

![hq720](https://github.com/user-attachments/assets/e7f0512e-9116-4159-8a90-5ad1b5cf5dc4)


"Guess a number" is a simple game where you try to guess the number, that your opponent (computer) has come up with.
The number is being randomly selected between 1 and 100.

You are prompted to enter your guess and will receive a hint if it's too low or too high.

If you enter not a valid number, you will get a message and prompt to try again.

You win when your number matches the number choosen by the computer and you will see congratulation message.

The messages are fun and engaging...or at least they are trying to be...
